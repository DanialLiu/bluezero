Detailed Description

template<typename TReq, typename TRep>
class b0::ServiceServer< TReq, TRep >

The service server template class.

This template class specializes b0::AbstractServiceClient to a specific request/response type.

If using a callback, when a request is received, it will be handed to the callback, as long as a spin method is called (e.g. Node::spin(), Node::spinOnce() or ServiceServer::spinOnce()).

You can directly read requests and write replies from the underlying socket, by using ServiceServer::poll(), ServiceServer::read() and ServiceServer::write().

See also
b0::ServiceClient, b0::ServiceServer, b0::AbstractServiceClient, b0::AbstractServiceServer

server_node.cpp, and server_node_object.cpp.
