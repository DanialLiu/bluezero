Detailed Description

template<typename TMsg>
class b0::Subscriber< TMsg >

The subscriber template class.

This template class specializes b0::AbstractSubscriber to a specific message type, and it implements the spinOnce method as well.

Important when using a callback: you must call b0::Node::spin(), or periodically call b0::Node::spinOnce(), otherwise no message will be delivered.

Otherwise, you can directly read from the SUB socket, by using Subscriber::read(). Note: read operation is blocking. If you do not want to block, use Subscriber::poll() first.

See also
b0::Publisher, b0::Subscriber, b0::AbstractPublisher, b0::AbstractSubscriber
Examples:
multi_subscriber.cpp, subscriber_node.cpp, subscriber_node_object.cpp, and subscriber_node_object_manual.cpp.
