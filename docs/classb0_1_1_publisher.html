Detailed Description

template<typename TMsg, bool notifyGraph = true>
class b0::Publisher< TMsg, notifyGraph >

The publisher class.

This class wraps a ZeroMQ PUB socket. It will automatically connect to the XSUB socket of the proxy (note: the proxy is started by the resolver node).
