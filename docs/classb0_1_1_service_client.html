Detailed Description

template<typename TReq, typename TRep>
class b0::ServiceClient< TReq, TRep >

The service client template class.

This template class specializes b0::AbstractServiceClient to a specific request/response type. It implements the call() method as well.

The remote service is invoked with ServiceClient::call() and the call is blocking. It will unblock as soon as the server sends out a reply.

You can make it work asynchronously by directly using ServiceClient::write(), and polling for the reply with ServiceClient::poll(), followed by ServiceClient::read().

b0::ServiceClient, b0::ServiceServer, b0::AbstractServiceClient, b0::AbstractServiceServer
Examples:
client_node.cpp, and client_node_object.cpp.
